Internet Connection Internet-yhteys

The Gateway needs an Internet connection. You can connect using either an Ethernet cable (RJ45) or via a wireless Wi-Fi network. To ensure the most reliable Internet connectivity, Ethernet is recommended.

Ruuvi Gateway tarvitsee toimiakseen internet-yhteyden. Voit yhdistää internetiin joko kiinteän (Ethernet RJ45) tai langattoman (Wi-Fi) lähiverkkoyhteyden avulla. Häiriöttömän yhteyden takaamiseksi suosittelemme käyttämään aina kiinteää yhteyttä.

Select connection type Valitse yhteystyyppi

You're accessing the Ruuvi Gateway configuration wizard from your local area network. For this reason, you cannot adjust the connection type. In order to change the connection type, please reconfigure the Gateway using its Wi-Fi hotspot. To start the hotspot, take a paper clip and press the pinhole button for a short time, found under the power socket. Olet avannut Ruuvi Gatewayn käyttöönottosivun lähiverkkoyhteyden kautta. Automatic Updates Automaattiset päivitykset

Please configure a convenient schedule for automatic firmware updates. During the update, your Ruuvi Gateway will download the latest firmware and then will automatically install it and reboot. (You may notice a 1-5 minutes downtime during the installation of the update.)

Valitse sinulle sopivin aika automaattisten ohjelmistopäivitysten toteuttamiselle. Päivityksen aikana Ruuvi Gateway lataa uusimman saatavilla olevan laiteohjelmiston itsenäisesti ja käynnistyy sen asennettuaan uudelleen. (Laite saattaa olla poissa käytöstä 1–5 minuutin ajan päivityksen aikana.)

Custom Server Mukautettu palvelin

Use your own and/or a third party server.

Käytä omaa ja/tai kolmannen osapuolen palvelinta.

 
 
 

  1. Subscribe to service messages from this Gateway:

    Tilaa palveluviestit tästä Gatewaysta:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/AA:BB:CC:DD:EE:FF/gw_status" -v

  2. Subscribe to service messages from any Gateway:

    Tilaa palveluviestit mistä tahansa Gatewaysta:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/#/gw_status" -v

  3. Subscribe to messages from a Bluetooth sensor with the MAC:

    Tilaa viestit tämän MAC-osoitteen sisältävästä Bluetooth-anturista:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/#/<TAG_MAC>/#" -v

  4. Subscribe to messages from any Bluetooth sensor heard by this Gateway:

    Tilaa viestit kaikista tämän Gatewayn kuulemista Bluetooth-antureista:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/AA:BB:CC:DD:EE:FF/#/#" -v

  5. Subscribe to all messages:

    Tilaa kaikki viestit:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/#" -v

How to reconfigure? Tarvitsetko aloittaa alusta?

If you misconfigured, you can restart the configuration Wi-Fi hotspot by taking a paper clip and briefly pressing the small pinhole button under the power socket.

Mikäli haluat käynnistää ohjatutn käyttöönoton uudelleen, paina laitteen virtaliittimen alla olevaa nollauspainiketta lyhyesti paperiliittimellä ja yhdistä sitten päätelaitteellasi uudelleen Ruuvi Gatewayn muodostamaan Wi-Fi-yhteyspisteeseen.

If you want to reset to the factory settings, take a paper clip and press the small pinhole button under the power socket for 10 seconds.

Mikäli haluat palauttaa tehdasasetukset, pidä paperiliittimellä virtaliittimen alla olevaa nollauspainiketta alaspainettuna kymmenen sekunnin ajan.
