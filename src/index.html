Internet Connection Internet-yhteys

Ruuvi Gateway requires an Internet connection. Connect an Ethernet (RJ45) cable to use a fixed network connection or connect to a local Wi-Fi network wirelessly. To ensure the most reliable Internet connectivity, Ethernet connection is recommended.

Ruuvi Gateway tarvitsee internet-yhteyden. Yhdistä internetiin joko kiinteän (Ethernet RJ45) tai langattoman (Wi-Fi) lähiverkkoyhteyden avulla. Häiriöttömän yhteyden takaamiseksi suosittelemme käyttämään aina kiinteää Ethernet-yhteyttä.

If you specify a folder name, it will first attempt to read the configuration file for this Ruuvi Gateway from <GW_MAC>.json (for example, AABBCCDDEEFF.json). If that file fails, an attempt will be made to read the overall configuration from gw_cfg.json. Jos määrität kansion nimen, tämän Ruuvi Gatewayn määritystiedot pyritään ensin hakemaan <GW_MAC>.json-tiedostosta (esimerkiksi AABBCCDDEEFF.json). Jos tiedoston lukeminen epäonnistuu, määritykset haetaan tiedostosta gw_cfg.json.

Also, the configuration file can be generated dynamically by the server. To do this the server must analyse the HTTP header 'ruuvi_gw_mac', which contains the MAC address in the format XX:XX:XX:XX:XX:XX. Lisäksi palvelin voi luoda konfigurointitiedoston dynaamisesti. Tätä varten palvelimen tulee analysoida HTTP-otsikko 'ruuvi_gw_mac', joka sisältää MAC-osoitteen muodossa XX:XX:XX:XX:XX:XX.

Automatic Updates Automaattiset päivitykset

Set a suitable schedule for installing automatic firmware updates. During the update, your Ruuvi Gateway will download the latest firmware and then will automatically install it and reboot. (You may notice a 1-5 minutes downtime during the update.)

Valitse sinulle sopivin aika automaattisten ohjelmistopäivitysten toteuttamiselle. Päivityksen aikana Ruuvi Gateway lataa uusimman saatavilla olevan laiteohjelmiston itsenäisesti ja käynnistyy sen asennettuaan uudelleen. (Laite saattaa olla poissa käytöstä 1–5 minuutin ajan päivityksen aikana.)

How to reconfigure? Kuinka voin muokata asetuksia?

If you need to modify these settings later, you can reconfigure your Ruuvi Gateway by restaring the Wi-Fi hotspot. Locate the pinhole button under the power socket and press the button briefly using a paperclip.

Mikäli haluat myöhemmin muokata Ruuvi Gatewayn asetuksia, voit käynnistää Wi-Fi-yhteyspisteen ja ohjatun käyttöönoton uudelleen painamalla lyhyesti laitteen virtaliittimen alla olevaa nollauspainiketta paperiliittimellä.

If you want to reset to the factory settings, use a paperclip and press the small pinhole button under the power socket for 10 seconds.

Mikäli haluat palauttaa tehdasasetukset, pidä paperiliittimellä virtaliittimen alla olevaa nollauspainiketta alaspainettuna kymmenen sekunnin ajan.
