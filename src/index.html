Internet Connection Internet-yhteys

Ruuvi Gateway requires an internet connection. Connect an Ethernet (RJ45) cable to use a fixed network connection or connect to a local Wi-Fi network wirelessly. To ensure the most reliable internet connectivity, Ethernet connection is recommended.

Ruuvi Gateway tarvitsee internet-yhteyden. Yhdistä internetiin joko kiinteän (Ethernet RJ45) tai langattoman (Wi-Fi) lähiverkkoyhteyden avulla. Häiriöttömän yhteyden takaamiseksi suosittelemme käyttämään aina kiinteää Ethernet-yhteyttä.

Automatic Configuration Download Automaattinen asetusten lataus

If you are an advanced user, you can set up Ruuvi Gateway router to fetch its configuration automatically from a remote server. In this case, you don't need to configure Ruuvi Gateway settings manually since Ruuvi Gateway will fetch settings file automatically. If this doesn't concern you, please continue to the next step.

Jos olet edistynyt käyttäjä, voit määrittää Ruuvi Gateway -reitittimen hakemaan asetukset automaattisesti etäpalvelimelta. Tästä on erityisesti hyötyä, jos sinulla on useita Ruuvi Gateway -reitittimiä tai sinulla ei ole pääsyä ohjattuun määrittämiseen lähiverkon kautta. Mikäli tämä ei koske sinua, voit jatkaa seuraavaan kohtaan.

Bluetooth Scanning Bluetooth-skannaus

Ruuvi Gateway continuously listens for Bluetooth advertisements from its surroundings. If you are an advanced user you can choose different listening options and set sensor filtering.

Ruuvi Gateway kuuntelee jatkuvasti ympäristön Bluetooth-majakkaviestejä. Jos olet edistynyt käyttäjä, voit muuttaa kuunteluasetuksia ja anturien suodatusasetuksia.

Advanced settings Edistyneet asetukset

Filtering Suodatus

Bluetooth sensor filtering applies to the listening settings selected above. Enabling this feature allows you to whitelist or blacklist sensors based on their MAC addresses. Bluetooth-anturin suodatus koskee yllä valittuja kuunteluasetuksia. Ottamalla tämän ominaisuuden käyttöön voit lisätä anturit sallittujen tai mustalle listalle niiden MAC-osoitteiden perusteella.

In this mode, the Ruuvi Gateway only relays data from the selected sensors and rejects all other Bluetooth devices or sensors it hears. Tässä tilassa Ruuvi Gateway välittää tietoja vain valituista antureista ja hylkää kaikki muut kuulemansa Bluetooth-laitteet tai anturit.

In this mode, the Ruuvi Gateway rejects data from the selected sensors and allows data from other Bluetooth devices or sensors according to the listening selection. Tässä tilassa Ruuvi Gateway hylkää valittujen antureiden lähettämät tiedot ja sallii tiedot muista Bluetooth-laitteista tai antureista kuunteluvalinnan mukaan.

Filtered sensors (or other Bluetooth devices) Suodatetut anturit (tai muut Bluetooth-laitteet)
