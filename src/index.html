Custom Server Settings Mukautetut palvelinasetukset

Use your own or a third party server.

Käytä omaa tai kolmannen osapuolen palvelinta.

  1. Subscribe to service messages from this gateway:

    Tilaa palveluviestit tästä yhdyskäytävästä:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/AA:BB:CC:DD:EE:FF/gw_status" -v

  2. Subscribe to service messages from any gateway:

    Tilaa palveluviestit mistä tahansa yhdyskäytävästä:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/#/gw_status" -v

  3. Subscribe to messages from the bluetooth tag with MAC:

    Tilaa viestit tämän MAC-osoitteen sisältävästä Bluetooth-anturista:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/#/<TAG_MAC>/#" -v

  4. Subscribe to messages from any bluetooth tags connected to this Gateway:

    Tilaa viestit kaikista tähän yhdyskäytävään liitetyistä Bluetooth-antureista:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/AA:BB:CC:DD:EE:FF/#/#" -v

  5. Subscribe to all messages:

    Tilaa kaikki viestit:

    mosquitto_sub -h test.mosquitto.org -p 1883 -t "ruuvi/#" -v

Internet connection Internet-yhteys

The Gateway needs an internet connection to send your data to a Cloud service. You can connect either via a Wi-Fi network or using an Ethernet cable (RJ-45).

Ruuvi Gateway käyttää internet-yhteyttä kerättyjen tietojen välittämiseen. Voit yhdistää internetiin joko WLAN-yhteyden tai Ethernet-kaapelin (RJ-45) avulla.
